## 📝 Инструкция по использованию

### **Для пользователя:**
1. Начните вводить номер телефона
2. Маска применится автоматически
3. Введите 11 цифр для полного номера
4. Система автоматически преобразует 8 в +7

### **Для разработчика:**
```javascript
import { formatPhoneMask, validatePhone } from '@/lib/phone-mask';

// Форматирование ввода
const formatted = formatPhoneMask('89611166603');
// Результат: +7(961)116-66-03

// Валидация
const isValid = validatePhone('+7(961)116-66-03');
// Результат: true
```
